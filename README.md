# AfriConnect Summit 2026

## À propos
Site vitrine complet pour AfriConnect Summit, une conférence tech panafricaine fictive réunissant développeurs, entrepreneurs et investisseurs du continent. Projet réalisé dans le cadre de l'examen de Technologie Web.

## Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, variables CSS, animations, media queries)
- JavaScript Vanilla (aucun framework, aucune librairie)
- Google Fonts (Space Grotesk + Inter)
- Bootstrap Icons
- Git & GitHub

## Fonctionnalités JavaScript implémentées
- Dark Mode / Light Mode avec persistance via `localStorage`
- Navbar dynamique (changement d'apparence au scroll)
- Menu hamburger fonctionnel en version mobile
- Animations au scroll via `IntersectionObserver`
- Onglets interactifs pour le programme (3 jours)
- Filtrage dynamique des intervenants par thématique
- Validation complète du formulaire d'inscription (regex email, longueur des champs, retours visuels)
- Compte à rebours en temps réel jusqu'à la conférence
- Compteurs animés des statistiques clés
- Bouton retour en haut de page
- Année dynamique dans le footer
